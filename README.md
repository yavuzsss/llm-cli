LLM CLI Chat
Komut satırından yapay zeka ile sohbet etmeyi sağlayan bir Python uygulaması.

Kurulum
Repoyu indir
.env.example dosyasını .env olarak kopyala ve Groq API anahtarını ekle
Kütüphaneleri yükle:
bash
python -m pip install -r requirements.txt
Çalıştırma
bash
python chat.py
Çıkmak için exit yaz.

Kullanılan Kütüphaneler
openai — API ile iletişim için
python-dotenv — API anahtarını güvenli saklamak için
